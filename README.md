# 📂 Gestor de Arquivos - O Jogo

Um Web App interativo para praticar a teoria de arquivos (Arquivologia).

## 🚀 Como jogar
1. O app apresenta o nome de um documento aleatório.
2. Você deve selecionar o *Setor* de proveniência.
3. Você deve selecionar o *Ciclo de Vida* (Teoria das Três Idades).
4. Você tem 10 segundos por rodada!

## 🛠️ Tecnologias
- Python
- [Flet](https://flet.dev) (Interface e Web)
